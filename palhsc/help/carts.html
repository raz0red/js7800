The different ways that cartridges can be loaded into JS7800 are detailed throughout the remainder of this tab. JS7800 should be compatible with the vast majority of cartridge ROMs available for the Atari 7800 (both original retail and homebrew releases).

Atari 7800 cartridge ROMs typically have a .bin or .a78 file extension. The .a78 extension is preferred as it contains header information that provides the emulator with information about how the particular cartridge should be interpreted. It is also worth noting that JS7800 does support cartridge ROMs that are contained within .zip files.

Cartridge Drop-Down Menu

The "Cartridge Drop-Down" menu (highlighted in red below) contains a list of Atari 7800 cartridges to select from.

Select Atari 7800 Cartridge...

By default, the cartridge list is populated with a selection of excellent homebrew titles. To load a cartridge, simply select the appropriate item in the menu.

Custom "cartridge lists" are also supported. For more information on creating and loading custom cartridge lists, refer to the Cartridge List GitHub page.

Select Local Files

The "Select Local File" button (highlighted in red below), is used to select and load an Atari 7800 cartridge ROM file that is located on the local device.

Select Atari 7800 Cartridge...

Once the button is pressed, a file chooser will be displayed that allows for the selection of a local file. By default, the file chooser limits the files displayed to .a78, .bin, .zip, and .json file extensions.

The local file chooser can also be used to select a "cartridge list" file (.json file extension). Selecting a "cartridge list" file will update the contents of the Cartridge Drop-Down Menu. For more information on cartridge lists, refer to the Cartridge List GitHub page.

Select Remote Files

The "Select Remote Files" button (highlighted in red below), is used to specify the URL of a remote Atari 7800 cartridge ROM.

Select Atari 7800 Cartridge...

Once the button is pressed, an input dialog will be displayed that allows for specifying the URL to a remote ROM file. For example, entering the following URL will cause the homebrew game, "bonQ", to be loaded:

http://raz0red.github.io/js7800/roms/bonq.a78

The remote file chooser can also be used to select a remote "cartridge list" file (.json file extension). Selecting a "cartridge list" file will update the contents of the Cartridge Drop-Down Menu. For more information on cartridge lists, refer to the Cartridge List GitHub page.

Drag and Drop Files

Dragging and dropping local Atari 7800 cartridge ROMs can be used as alternative to selecting files via the local file browser (see Select Local Files).

As demonstrated above, local ROM files can be loaded by simply dragging and dropping them onto the emulator from File Explorer on Windows or Finder on the Mac.

In addition to cartridge ROM files, "cartridge list" files (.json file extension) can also be dragged and dropped onto the emulator. Dragging and dropping a "cartridge list" file will cause the contents of the Cartridge Drop-Down Menu to be updated. For more information on cartridge lists, refer to the Cartridge List GitHub page.

Drag and Drop Links

Dragging and dropping links that point to remote Atari 7800 cartridge ROMs can be used as alternative to specifying URLs via the remote file chooser (see Select Remote Files).

As demonstrated above, ROMs can be loaded by simply dragging and dropping remote file links onto the emulator. In the specific example above, a link is being dragged from the AtariAge Atari 7800 Programming Forum, a great source for both in development and completed homebrew titles. It is also worth noting that some links on the AtariAge forum require the dragging to be performed via the "download count" link versus the game title link (as shown above).

In addition to cartridge ROM file links, "cartridge list" file links can also be dragged and dropped onto the emulator. Dragging and dropping a "cartridge list" file link will cause the contents of the Cartridge Drop-Down Menu to be updated. For more information on cartridge lists, refer to the Cartridge List GitHub page.

Request Parameters

JS7800 supports the following set of request parameters. Request parameters can be added to the JS7800 URL to control various emulator options. For example, the cart request parameter can be used to force loading of a specific cartridge when JS7800 is initially displayed.

cart
The cart request parameter can be used to load a specific cartridge when the emulator page is initially loaded. The following URL will cause the homebrew game, "bonQ", to load:

https://raz0red.github.io/js7800/?cart=http://raz0red.github.io/js7800/roms/bonq.a78

cartlist
The cartlist request parameter can be used to load a cartridge list into the Cartridge Drop-Down Menu when JS7800 is initially loaded. The following URL will cause a list of homebrew demos to be populated in the cartridge drop-down menu:

https://raz0red.github.io/js7800/?cartlist=http://raz0red.github.io/js7800/roms/romlist-homebrew-demos.json

For more information on cartridge lists, refer to the Cartridge List GitHub page.

Multiple request parameters can be combined to set more than one option when JS7800 is loaded. The following URL will load a specific game, "bonQ", and a cartridge list containing homebrew demos:

https://raz0red.github.io/js7800/?cart=http://raz0red.github.io/js7800/roms/bonq.a78&cartlist=http://raz0red.github.io/js7800/roms/romlist-homebrew-demos.json

Since adding request parameters will typically result in long URLs, the use of the GitHub URL shortening service, git.io, is recommended. The following URL is the shortened version of the example above, containing both cart and cartlist request parameters:

https://git.io/JfcmO
