Controls Bar

The "controls bar", which is located directly beneath the JS7800 emulator screen, is used to control various aspects of emulation, including access to settings and help dialogs.

The buttons and controls that comprise the controls bar are described below.

Pauses emulation of the current cartridge. Once paused, this button can be used to resume emulation.
Toggles on and off sound output.
Restarts the current cartridge. This is equivalent to performing a reload and clean start of the current cartridge.
The select button represents the physical "SELECT" button that is present on the Atari 7800 console. The behavior of this button is cartridge-specific, but is typically used to allow for selection of the game's current settings (difficulty levels, etc.).
The reset button represents the physical "RESET" button that is present on the Atari 7800 console. The behavior of this button is cartridge-specific, but is typically used to end the current game (if one is in progress).
Represents the physical left difficulty switch that is present on the Atari 7800 console. The behavior of this switch is cartridge-specific.
Represents the physical right difficulty switch that is present on the Atari 7800 console. The behavior of this switch is cartridge-specific.
Opens the JS7800 Global Leaderboard page. This page provides the ability to view global high scores and top players. The initial set of scores displayed in the leaderboard defaults to the game that is currently being played in JS7800 (if it supports global high scores). For more information on the global leaderboard, refer to the "High Scores" tab on the left. A raw dump of supported games and variants (identified by cartridge MD5 hash without header) can be found here.
Displays this "Help" dialog.
Displays the "Settings" dialog. See the "Settings Dialog" help tab on the left for detailed information.
Toggles between full screen and non-full screen modes.
